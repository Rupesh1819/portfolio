# Rupesh Sanjay Shete — AI Engineer Portfolio 🧠✨

A premium, high-performance portfolio built for an elite AI Engineer. Designed with fluid animations, interactive components, and a modern "Neural Cyberpunk" aesthetic to showcase machine learning projects, architectures, and research.

## 🌟 Key Features

- **Interactive Glass Portrait**: A premium hover-reactive Hero portrait that smoothly transitions from grayscale to vibrant color on hover.
- **Magnetic Custom Cursor**: A framer-motion powered pointer with `mix-blend-difference` that intelligently scales on interactable elements.
- **Dynamic AI Architectures**: Custom visual diagrams tailored to specific AI systems (Autonomous RAG Pipelines, Multi-Agent Orchestration).
- **Project Case Studies**: Dynamic routing for detailed project breakdowns featuring metrics, tech stacks, and direct links to live deployments.
- **Cinematic Polish**: Staggered "split text" reveals, animated background particle effects (`BackgroundFX`), and premium glassmorphism (`glass-panel`).
- **Performance First**: Built on Next.js 16 (App Router) with Turbopack, Tailwind v4, optimized font loading, and pristine accessibility scores.

## 🛠️ Technologies & Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (React 19)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling**: [Lenis](https://studiofreight.github.io/lenis/)
- **Forms**: [React Hook Form](https://react-hook-form.com/)
- **Email**: [Resend](https://resend.com/)

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Architecture Highlights

- `/src/app`: Next.js App Router (pages, layouts, dynamic `[slug]` routes).
- `/src/components/sections`: Modular sections (Hero, About, Projects, Experience, Skills, Architectures, Contact).
- `/src/components/ui`: Reusable interactive components (CustomCursor, BackgroundFX, SmoothScroll).
- `/src/lib/data.ts`: The single source of truth for all portfolio content.
- `/src/types/index.ts`: Strict TypeScript interfaces for projects and experiences.

## 🌐 Deployment
